
Project that allows to create customers.

## Tecnologies and libs

- **[Angular Framework](https://angular.io/)**
- **[Angular Material Design](https://material.angular.io/)**

### Features
- Create, edit and delete a customer.
- List all customers.
- Search a customer by name
- Create, edit or delete a comment to a customer.

## Run Locally

- Clone the project

```bash
  git clone https://github.com/luisscarlos/customer-register-web.git
```

- Enter folder project: `cd customer-register-web`
- Install libs with  `npm install`
- Start the application with `ng serve -o`
PS: To frontend application run properly with all functionalities, is needed [backend](https://github.com/luisscarlos/customer-register-api) is up first.
